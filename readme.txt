== Description==

Extended widgets addon kit for Elementor for creating accordion post and radial gauge. Animated gauge using gauge.js library


Features

<ui>

        * 100% Responsive and interactive
 	* Use anywhere in post and pages
        * Elementor Widgets for live editing of gauge,accordion post
        * Using Open source gauge.js Javascript Library
        * For multiple gauge/charts on same page, use unique id/chart id
        * Canvas based charts

</ui>


== Installation ==
**Method 1:**

On your WordPress admin dashboard
Visit ‘Plugins > Add New’
Search for ‘extended widgets addon kit for Elementor’ and install the plugin.
Activate ‘extended widgets addon kit for Elementor’ from your Plugins page.

**Method 2:**

Download the plugin from WordPress.org repository
On your WordPress admin dashboard, go to ‘Plugins> Add New> Upload Plugin’.
Upload the downloaded plugin file (extended widgets addon kit for Elementor.zip) and click ‘Install Now’
Activate ‘extended widgets addon kit for Elementor’ from your Plugins page.

Once the plugin is activated, you can create gauge charts and accordion post through
through elementor widgets panel.



== Frequently Asked Questions ==

= How can I generate a gauge? =

You can generate gauge chart through Elementor widget. 


= Can I display more than one gauge chart on the same page? =

Yes, you can create unlimited charts. Use unique chart ID for multiple charts on same page.
